# bs: Build system based on executables
This project uses [jaandrle/bs: The simplest possible build system using executable/bash scripts](https://github.com/jaandrle/bs).

## Available executables

### bs/build.sh
Linted project and when no errors, build it (see `bs/build/*`).

### bs/examples.sh
Lists examples and runs them as tests.

### bs/lint.sh
Linted project using jshint and check project size.

### bs/build/cjs.sh
Builds CommonJS version of the project.

### bs/build/docs.sh
Generates docs using typedoc.
